

1)  CSS 中 zoom:2 与 transform: scale(2)有何区别？
    > zoom 的缩放是相对于左上角的；而 scale 默认是居中缩放；
    > zoom 的缩放改变了元素占据的空间大小；而 scale 的缩放占据的原始尺寸不变，页面布局不会发生变化；
    > zoom 会回流,重新 layout,性能慢.

12) 什么叫做排序算法的稳定性？它有什么作用？
    > 排序算法的稳定性指的是数值相同的元素在排序前后是否保持相同的顺序.如果位置不发生改变则称为稳定的排序算法,否则则为不稳定的.
    > 如果要根据多个属性排序,进行主要排序与次要排序,需要使用稳定排序

14) try catch finally 语句一般在什么情况下使用？有哪些需要注意的问题？最佳实践是什么？
    > JS 读取一行执行一行,在遇到错误时会直接报错无法继续运行.在函数有可能运行异常时,接收并处理错误,使代码能够继续运行.
    ```
        try{
    //要监控的代码
    } catch(err){
    //处理异常 throw
    console.error(err);
    } finally{
    //无论如何都会执行
    }
    ```
15) 简述你对面向对象的三大特性的理解（不是把三大特性列出来）
    > 封装,将复杂的操作,计算打包成一个一个的函数,在写代码时直接调用这些打包好的函数,从而从更高的层次看待要解决的问题,使用时可以不用关注底层细节.也指把数据和函数打包放在对象里的过程.
    > 多态,多个不同的对象拥有相同的一组接口.(相同的方法,方法的函数名称,参数类型及顺序,返回值的类型).某段代码在运行的时候只需要对象有这些相同的接口即可正常工作,而不关注对象的具体类型
    > 继承,当一个类型拥有另一个类型的所有或大多数特性时,可以使用继承的方式让其直接拥有另一个类型的所有属性和方法.被继承者称为父类,继承者称为子类.
